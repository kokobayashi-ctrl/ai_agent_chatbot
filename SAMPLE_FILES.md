# サンプルファイル使用例

このフォルダには、アプリケーションの動作を確認するためのサンプルファイルを配置してください。

## ファイル配置例

### data/company/
会社に関する情報を含むファイル:
- 会社概要.pdf
- 組織図.docx
- 会社沿革.txt

### data/service/
サービスに関する情報を含むファイル:
- サービス説明.pdf
- 料金プラン.docx
- FAQ.txt

### data/customer/
顧客とのやりとりに関する情報を含むファイル:
- 問い合わせ履歴.pdf
- 対応マニュアル.docx
- クレーム対応事例.txt

## ファイルアップロード機能のテスト方法

1. アプリケーションを起動: `streamlit run main.py`
2. 画面上部の「問い合わせに関連するファイルをアップロード」エリアにファイルをドラッグ&ドロップ
3. チャット欄で質問を入力（例: "このファイルの内容を要約してください"）
4. AIがアップロードされたファイルの内容を基に回答を生成

## テスト用質問例

### 既存データベースを使用する場合（data/フォルダ）
- "会社の概要を教えてください"
- "サービスの料金プランについて教えてください"
- "過去の問い合わせ対応事例を教えてください"

### アップロードファイルを使用する場合
- "アップロードしたファイルの要点をまとめてください"
- "この資料の中で重要なポイントは何ですか？"
- "ファイル内の○○について詳しく教えてください"
